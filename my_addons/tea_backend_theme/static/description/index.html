How to Custom

  • Custom Login Page Background Picture

    Found in the /static/src/js/login.js file 
    $.backstretch([
    "/tea_backend_theme/static/src/img/bg/1.jpg",
    "/tea_backend_theme/static/src/img/bg/2.jpg",
    "/tea_backend_theme/static/src/img/bg/3.jpg",
    "/tea_backend_theme/static/src/img/bg/4.jpg"
    ], {
        fade: 1000,
        duration: 8000
        }
    );

  • Custom Navbar Colour

    Found in the /static/src/less/variables.less file 
    @odoo-brand-primary: #4F5A67;

  • Custom Sidebar Colour

    Found in the /static/src/less/sidebar.less file 
    @odoo-sidebar-bg: #5E6672;
